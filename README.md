# chatbot-system
•	Built a voice-enabled hospital chatbot system that assists users in booking, updating, and rescheduling appointments using real-time speech recognition and text-to-speech feedback. 
•	Integrated OpenAI’s GPT-3.5 Turbo API for intelligent responses and QR code generation for appointment confirmation. Implemented using HTML, CSS, JavaScript (Web Speech API), and Flask with local data storage and Excel export functionality
